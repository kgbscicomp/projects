For McGurk V3: just run the mcgurkV3_arr_generator_Ver2.m and an excel file will be saved in your subject's folder. 

For McGurk V2: 
1) Run the mcgurkV2_arr_generatorPart1_Ver2.m 
2) Open the variable "conditions"
3) Check the corresponding subject id on the excel sheet named ECOG_McGurk_V2.xls and copy-paste the labels into the variable "conditions"
4) Run phoneme_compare.m
5) Run mcgurkV2_arr_generatorPart2_Ver2.m
6) An excel file will be saved in your subject's folder.

For sub 1124: 
1)Follow steps 1,2,3 above. 
2) Run phoneme_compare_1124.m
3) Run mcgurkV2_arr_generatorPart2_1124.m
4) An excel file will be saved in your subject's folder.
